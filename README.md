# projeto-blog-django-25
 Projeto de blog utilizando django

# Ferramentas utilizadas

 - Python
 - Django
 - Docker


# Passa a passo

Criar ambiente virtual python

```
python -m venv venv
```


```
venv/Scripts/Activate
```

```
python -m pip install --upgrade pip
```

```
pip install django
```

```
mkdir djangoapp
```

```
cd djangoapp
```

```
django-admin startproject project .
```
